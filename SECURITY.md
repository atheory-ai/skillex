# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for a suspected security problem.

Report vulnerabilities privately to:

- `jeremy@atheory.ai`

Include:

- a clear description of the issue
- affected versions or commit range if known
- reproduction steps or a proof of concept if available
- any suggested mitigation

I will review reports in good faith and work toward a fix before public disclosure.

## Supported Versions

Security fixes are best-effort and are expected to target the current `main` branch first.

At this stage of the project, older releases may not receive backported fixes.
